<Steps>
<Step>

#### Access Active Pages

Navigate to your `Active Pages` list to view all your current fundraising pages.

</Step>
<Step>

#### Select Edit

Tap the edit icon (pencil) next to the page you want to modify.

</Step>
<Step>

#### Update Page Details

You can edit the following fields:

- `Banner` - Change the header image for your page
- `Avatar` - Update your profile picture
- `Title` - Modify your fundraiser name
- `Target` - Adjust your fundraising goal amount
- `Description` - Update your cause explanation
- `Page Close Date` - Change when your fundraiser ends
- `Theme` - Select a different visual style

<Callout type="warning">
**Important Restrictions**

You cannot edit causes or events after page creation. If you need to change these, you must create a new fundraising page.
</Callout>

</Step>
<Step>

#### Save Changes

Tap the `Update Page` button to save your modifications. Your changes will be applied immediately.

</Step>
</Steps>
